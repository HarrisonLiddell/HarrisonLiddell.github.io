Web Development

I have had 3/4 years experience programming websites using HTML, CSS and Javascripts and most of the teaching for this i have done myself with a bit of help from college lessons. My website development process starts by taking the all the graphical elements (colours, fonts, layout, images) defined in the design process and coding them using web industry standards (HTML5, CSS3, JS). This is typically done by first coding the index/home page, followed by templates for the other pages that the client has requested, after doing this I would start to add all the Javascript elements in once I am happy with the stylesheets. Then finally i would make sure that the site works on most site engines and devices using live servers and inspect elements. When the website is ready for review i would upload the site to a github page so that the client can virtually see that the page is working and up to there standard.

All of the websites that I have created have been built using Visual studio code and have all been made by my own coding techniques and templates.
